## 一、我们如何收集和使用个人信息

我们遵循合法、正当、必要原则，仅在实现本应用功能与服务所必需的范围内处理个人信息。不同功能可能需要不同信息，具体以页面提示与系统授权弹窗为准。

### 1. 账户注册与登录

为完成账户登录与身份验证，我们可能处理以下信息：

1. **微信登录**  
   当您选择微信登录时，我们将通过微信开放平台获取您的账号相关信息（如昵称、头像、OpenID/UnionID），用于登录验证与账户识别。

2. **Apple 登录（仅 iOS）**  
   当您在 iOS 端选择“Sign in with Apple”登录时，我们将获得 Apple 提供的用户标识符；若您授权，可能同时获得姓名与邮箱（含 Apple 提供的转发邮箱）。上述信息仅用于登录验证与账户识别。

### 2. 教务数据展示与学习服务

为向您展示课程、考试、成绩等学习信息，本应用可能在您操作下从学校官方系统页面获取并在本地生成相关数据，常见数据类型包括：

- 课程表信息（课程名称、时间、地点、教师等）
- 考试信息（时间、地点、座位号等）
- 成绩与绩点信息（如学校支持）
- 校内日历/校内事件信息（如学校支持）

除您主动使用“共享课表”等需要云端处理的功能外，我们不会将上述学习数据上传至我们的服务器。

### 3. 共享课表

当您主动发起课表共享时，我们将把您选择共享的课表数据存储于服务器，并通过分享码提供他人访问。共享由您主动触发，您可随时停止共享或重新生成分享码。分享码有效期与访问规则以应用内说明为准。

### 4. 日历功能与系统日历权限

当您使用日历相关功能并授权日历权限后，本应用可能：

- 读取系统日历事件，用于与节假日、校内事件等信息合并展示；
- 向系统日历写入、更新或删除由本应用导出的课程/考试事件，以实现导出与管理。

我们不会因日历功能将您的系统日历事件内容上传至我们的服务器。

### 5. 通知与提醒

当您授权通知权限并启用提醒设置后，我们会向您发送与课程、考试、日历事项相关的通知提醒。

### 6. 意见反馈

当您提交意见反馈时，我们会处理您主动提交的信息，包括：

- 反馈内容（文字）
- 您选择提供的联系方式（如有）
- 您主动上传的附件（图片/视频）
- 为定位问题所必需的基础信息（如平台类型、应用版本号、构建号、时间、必要的错误信息）

反馈附件将存储在我们的服务器上，用于处理与留存反馈记录；当反馈被删除或按规则清理时，我们将删除相应附件。

### 7. 设备信息、应用信息与日志信息（含软件列表、设备 MAC、传感器类型等）

为保证应用功能的正常运行、完成必要的安全防护以及提升整体稳定性，本应用在您同意隐私政策后可能会根据系统权限及设备环境，读取以下信息：

- 设备运行环境信息：包括设备型号、系统版本、系统 API Level、应用安装来源、应用版本、设备唯一标识符（如 Android ID）、网络类型等，用于保证服务兼容性、错误定位与运行稳定性。

- 设备网络标识信息（如 MAC 地址）：该类信息仅用于网络环境校验、统计分析准确性校准、防止恶意攻击与异常设备识别。本应用不会使用该信息进行位置跟踪、设备画像、跨应用关联或广告定向。

- 应用安装情况（软件列表）：仅在涉及账号异常校验、作弊防护、登录环境安全判断等场景下，由第三方统计服务 SDK 读取“可见应用列表”，该过程遵循“最小必要”原则，不会对列表内容进行分析、推断或画像，更不会用于营销用途。

- 传感器类型列表：部分系统 API 或第三方基础能力（如统计 SDK、防作弊 SDK 或系统能力初始化）可能在您同意后读取设备支持的传感器类型列表，以判断设备能力并保障基础功能的稳定性。本应用不会对传感器数据进行采集分析，不会读取加速度、陀螺仪、步数、心率等传感器的实时数据。

上述设备环境信息仅用于:

- 保障应用基础能力的兼容性、稳定性与安全性；
- 防止恶意攻击与异常环境登录，提高账号安全性；
- 准确统计应用运行情况并用于产品改进；
- 判断第三方服务接口是否可用；
- 错误分析与问题定位（如崩溃原因调查）。

处理方式与范围:

- 本应用及相关第三方 SDK 均按照“合法、正当、最小必要”的原则处理上述信息；
- 信息仅在本地或传输至经授权的统计分析服务端进行匿名化处理；
- 本应用不会基于上述信息进行广告推送、用户画像或跨应用关联；
- 您可在系统权限管理中关闭相关权限，关闭后不影响大部分功能的正常使用。

本应用不会在您拒绝隐私政策前读取或处理上述信息；  
所有处理行为均在您点击“同意”并已授权系统权限后才会生效。

### 8. 位置信息（在您使用相关功能时）

当您使用需要定位的功能（如校园地图）并授权定位权限后，我们会处理位置信息以实现该功能。您可在系统设置中撤回授权，撤回后定位相关功能不可用。

### 9. 文件上传（如您使用相关功能时）

如您使用本应用提供的文件上传/资料相关功能，我们会处理您主动上传的文件内容及必要的文件信息（如文件名、大小、类型、上传时间），用于实现该功能。存储期限与删除规则以应用内说明为准。

## 二、个人信息的共享、转让、公开披露

我们不会出售、出租或以其他方式向第三方提供您的个人信息。仅在以下情形下，可能发生共享、转让或公开披露：

- 获得您的明确同意或授权；
- 为实现功能所必需，与服务提供方进行必要共享，并要求其承担相应的数据保护义务；
- 根据法律法规规定或监管部门要求；
- 为保护国家安全、公共安全、他人生命财产等重大合法权益所必需；
- 对外提供经去标识化处理且无法识别个人身份的信息，用于统计分析与研究。

## 四、个人信息的存储与保护

### 1. 存储地点

除法律法规另有规定或获得您的明确授权外，我们在中华人民共和国境内存储个人信息。

### 2. 存储期限

我们仅在实现目的所必需的期限内保存个人信息，超过期限将删除或匿名化处理，法律法规另有要求的除外。

- 学习数据通常存储于本地，用于展示与提醒；
- 共享课表数据在共享期间按功能需要存储于服务器，停止共享或到期后按规则删除或失效；
- 反馈与附件按反馈处理与留存需要保存，删除反馈或按规则清理时删除相应数据。

### 3. 安全措施

我们采取合理的安全措施保护个人信息安全，包括传输加密、访问控制与最小权限等。

您依法享有以下权利（以适用法律为准）：

- 访问与更正：在应用内查看并修改部分信息与设置；
- 删除：清除本地数据；对服务器侧数据可通过联系邮箱申请删除（以身份核验与法律法规要求为前提）；
- 撤回同意：在系统权限管理中撤回通知、定位、日历、相册/媒体等授权；
- 注销：如提供注销入口或通过邮箱申请注销，我们将在核验后处理，并依法删除或匿名化处理相关信息。
